How to get started with open source

How to start contributing to open source

My experience could be summarized into the following steps:

  1. Start with the document to understand how it is supposed to work (by reading the guides / document)

  2. notice the improvement you want to make, usually when you are using it

  3. go through the open issues and pull requests to make sure nobody was already working on it. If the changes require a fair amount of work, do NOT start working on it without discussing your idea with the core team members

  4. locate the exact place where you need to update in the source code. This may require a lot of effect.

  5. add relevent tests if necessary and update the code; make sure all the tests still pass

  6. send PR with a description of why you think it’s a good idea.
